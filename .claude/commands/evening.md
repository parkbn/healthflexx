# Evening Review

Good evening, Dr. Park. Run through the following steps in order:

## 1. Today's Calendar Recap

Use the Google Calendar MCP tool to fetch today's events. Display a summary of what was on the schedule:

```
TIME        EVENT              STATUS
──────────────────────────────────────
8:00 AM     Example Meeting    ✓ Done
10:30 AM    Example Call       ✓ Done
```

If no calendar tool is available, ask me to describe how the day went.

## 2. Task Review

Check the task list for all tasks. Sort them into three groups:

**Completed Today:**
- [x] Task description

**Still In Progress:**
- [ ] Task description

**Not Started:**
- [ ] Task description

If all tasks are completed, say "Clean sweep — everything done today."

## 3. Wins & Blockers

Ask me two questions, one at a time:

> What went well today? Any wins worth noting?

After I respond, then ask:

> Did anything block you or not go as planned?

## 4. Tomorrow Prep

Use the Google Calendar MCP tool to fetch tomorrow's events. Display them as a preview:

```
TOMORROW'S SCHEDULE
─────────────────────────────
8:00 AM     Example Meeting
2:00 PM     Example Call
```

Then ask:

> Based on today, is there anything you want to carry forward or add to tomorrow's priorities?

After I respond, create any new tasks so they're ready for the morning briefing.

## 5. Incomplete Task Triage

For any tasks still marked as in_progress or pending, ask me what to do with each one:

- **Carry forward** — keep it for tomorrow
- **Defer** — remove it from the active list for now
- **Drop** — mark it as completed / no longer needed

## Formatting

Keep the review concise and scannable. Use this structure for the final output:

```

[calendar recap]

📋 TASK SCORECARD
[completed / in-progress / not started counts]
[task details]

🏆 WINS
[waiting for input]

🚧 BLOCKERS
[waiting for input]

📅 TOMORROW PREVIEW
[tomorrow's calendar]

🔄 CARRY FORWARD
[triaged tasks]
```
