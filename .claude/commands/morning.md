# Morning Briefing

Good morning, Dr. Park. Run through the following steps in order:

## 1. Today's Calendar

Use the Google Calendar MCP tool to fetch today's events. Display them in a clean timeline:

```
TIME        EVENT
─────────────────────────────
8:00 AM     Example Meeting
10:30 AM    Example Call
```

If no calendar tool is available, ask me to paste or describe today's schedule.

## 2. Yesterday's Incomplete Tasks

Check the task list for any tasks that are still pending or in_progress. Display them grouped by status:

**Still In Progress:**
- [ ] Task description

**Not Started:**
- [ ] Task description

If the task list is empty or all tasks are completed, say "All clear — nothing carried over from yesterday."

## 3. Set Today's Big 3

Ask me to choose my **Big 3 priorities** for today. These are the three most important things I want to accomplish. Prompt me like this:

> What are your Big 3 priorities for today? These should be the tasks that, if completed, would make today a success.

After I respond, create each one as a task using the task tools so we can track them throughout the day.

## Formatting

Keep the briefing concise and scannable. Use this structure for the final output:

```
☀ MORNING BRIEFING — [Today's Date]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📅 TODAY'S SCHEDULE
[calendar timeline]

📋 CARRIED OVER
[incomplete tasks or "All clear"]

🎯 BIG 3 PRIORITIES
[waiting for input]
```
