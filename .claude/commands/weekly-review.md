# Weekly Review

Good afternoon, Dr. Park. Let's review the week. Run through the following steps in order:

## 1. This Week's Calendar Overview

Use the Google Calendar MCP tool to fetch all events from Monday through today. Display them grouped by day:

```
MONDAY 2/16
  9:00 AM     Example Meeting
  2:00 PM     Example Call

TUESDAY 2/17
  10:00 AM    Example Workshop
  ...
```

After displaying, summarize:
- Total number of meetings/events
- Busiest day
- Any days with no events

If no calendar tool is available, ask me to describe how the week went.

## 2. Task Scorecard

Check the task list and compile a full accounting of all tasks from the week:

```
WEEKLY SCORECARD
━━━━━━━━━━━━━━━
✅ Completed:    X
🔄 In Progress:  X
⬜ Not Started:  X
───────────────
   Total:        X
   Completion:   X%
```

Then list each group with task details.

## 3. Big 3 Retrospective

Review each day's Big 3 priorities (if they were set via /morning). For each one, note whether it was completed. Summarize:

> **Hit rate:** X out of Y Big 3 items completed this week.

If no Big 3 data is available, skip this section.

## 4. Wins of the Week

Ask me:

> What were your top wins this week? What are you most proud of?

## 5. Challenges & Lessons

After I respond to wins, ask:

> What challenged you this week? Anything you'd do differently?

## 6. Next Week Preview

Use the Google Calendar MCP tool to fetch events for next Monday through Friday. Display them as a preview:

```
NEXT WEEK AT A GLANCE
━━━━━━━━━━━━━━━━━━━━━
Monday:     X events
Tuesday:    X events
Wednesday:  X events
Thursday:   X events
Friday:     X events
```

Flag any notably busy or empty days.

## 7. Set Next Week's Priorities

Ask me:

> Based on this week's review and next week's calendar, what are your top 3 priorities for next week?

After I respond, create each one as a task so they're ready for Monday's morning briefing.

## 8. Incomplete Task Triage

For any tasks still marked as in_progress or pending, ask me what to do with each one:

- **Carry forward** — keep it for next week
- **Defer** — remove it from the active list for now
- **Drop** — mark it as completed / no longer needed

## Formatting

Keep the review concise and scannable. Use this structure for the final output:

```
📊 WEEKLY REVIEW — Week of [Date]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📅 THIS WEEK'S CALENDAR
[daily breakdown]

📋 TASK SCORECARD
[completion stats]

🎯 BIG 3 HIT RATE
[retrospective or skipped]

🏆 WINS
[waiting for input]

💡 LESSONS
[waiting for input]

📅 NEXT WEEK PREVIEW
[calendar preview]

🎯 NEXT WEEK'S PRIORITIES
[waiting for input]

🔄 CARRY FORWARD
[triaged tasks]
```
